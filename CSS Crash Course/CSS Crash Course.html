CSS Crash Course

Types of CSS

  • Inline - Style directly in the HTML element
  • Internal - Use <style> tag within the HTML file
  • External - Use a CSS file

Common Properties

  • color - Sets the text color. Use RGBa to set color and opacity.
  • background-color - Sets the background color. Use RGBa to set color and opacity.
  • width - Sets the width of an element
    • NEVER set hardcode unless absolutely necessary.
    • Set to 100%. Will resize automatically.
    • <max-width> will help with horizontal re-sizing to a specific size.
  • height - Sets the height of an element
    • NEVER set hardcode unless absolutely necessary.
    • Set to 100vh or vw. Will resize automatically.
    • <max-height> will help with vertical re-sizing to a specific size.
  • border - Sets the border of an element
    • ALWAYS use the short-hand version: border: "size"px solid "color"
  • padding - Sets the space between the content and the border of an element
    • Use the short-hand version: padding: "size"px
  • margin - Sets the space outside the border of an element
    • Use the short-hand version: margin: "size"px
  • display - Sets how an element is displayed
    • Use the short-hand version: display: "type"
  • font-size - Sets the size of the font
  • font-weight - Sets the weight (boldness) of the font
  • font-family - Sets the font family for text. Use Google fonts.

Specificity

Definition: The prioritization of the styling. Think - order of operations.
  • Class: Helps target what we want. Type a "." in front of the class to target the class, not HTML element. Classes have a higher priority than just HTML elements.
  • ID: A stronger-priority. Can only have one "ID" per HTML. Can have multiple in CSS.
  • !important: Highest/Stronges-priority. Do NOT use unless absolutely necessary.

Order of specificity (lowest to highest):

  • Tag Name:
    • Example: <div> <forward slash (/)div>
    • CSS: div {color: red;}
  • Class
    • Example: <div class="red"> <forward slash (/)div>
    • CSS: .red {color: red;}
  • ID
    • Example: <div id="home"> <forward slash (/)div>
    • CSS: #home {color: red;}
  • !important
    • Example: <div class="red"> <(/)div>
    • CSS: .red {color: red !important;}

Box Model

The box model is a way to visualize the structure of an element in CSS. It consists of the content, padding, border, and margin. Order is very important.

  • Content: The actual content of the element.
  • Padding: Space between the content and the border.
  • Border: The line surrounding the padding.
  • Margin: Space outside the border.

Complex Properties

  • Display
    • block - Takes up the full width of the parent element. Starts on a new line.
      • Examples:
        1. <div> </div> or;
        2. <h1> </h1>
    • inline - Takes up only the width of the element. Does not allow for adjustments in height or width. Does not start on a new line.
      • Examples:
        1. <span> </span> or;
        2. <a> </a>
  • Position
    • static - Default position. Elements are positioned according to the normal flow of the document.
    • relative - Positioned relative to its normal position. Can be moved using top, right, bottom, and left properties.
    • absolute - Positioned relative to the nearest positioned ancestor (not static). Can be moved using top, right, bottom, and left properties.
    • fixed - Positioned relative to the viewport. Does not move when scrolling.
    • sticky - A hybrid of relative and fixed. It behaves like relative until a certain point, then becomes fixed.

Media Queries

Media queries are used to apply different styles based on the device's characteristics, such as screen size or orientation. They allow for responsive design.

Common Breakpoints:

  • 576px - Mobile devices
  • 768px - Tablets
  • 992px - Small laptops
  • 1200px - Desktops

See Media Queries for more details.

Flexbox Layout

Flexbox is a layout model that allows for responsive design and easy alignment of elements. It provides a more efficient way to lay out, align, and distribute space among items in a container.

Use 'display: flex;' to make the adjustments.

  • flex-direction
    1. row
    2. column
  • align-items - vertical aligntment
    1. flex-start
    2. flex-end
    3. center
    4. stretch
  • justify-content - horizontal alignment
    1. flex-start
    2. flex-end
    3. center
    4. space-between
    5. space-around

Justify-content and align-items are used together!

If flex-direction is set to column, justify-content and align-items switch roles.
